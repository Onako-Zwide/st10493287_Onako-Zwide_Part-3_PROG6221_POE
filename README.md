# Project Description

CyberSecurity Chatbot is a WPF desktop application developed in C# using .NET 8.0. The project was completed over three parts and aims to educate users about cybersecurity through interactive conversations and useful cybersecurity tools. The application combines chatbot interaction, sentiment detection, memory features, task management, a cybersecurity quiz, NLP simulation, and activity logging into one integrated system.

---
# Features

## Part 1 Features

- GUI-based chatbot using WPF.
- Voice greeting on startup.
- Personalized greeting.
- Interactive cybersecurity conversations.
- Cybersecurity awareness topics.

  
## Part 2 Features

- Keyword recognition.
- Sentiment detection.
- Context-aware follow-up responses.
- User memory system.
- Returning-user recognition.
- Expanded cybersecurity topics.

  
## Part 3 Features

### Task Assistant with Reminders

- Add tasks.
- View tasks.
- Mark tasks as complete.
- Delete tasks.

### Cybersecurity Quiz

- 10 or more questions.
- Immediate feedback.
- Score tracking.
- Final score and performance message.- Optional reminders.
- Persistent JSON storage using tasks.json.
